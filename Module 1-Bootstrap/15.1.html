Image with Borders and Rounded Corners

This image demonstrates various border and rounded corner utilities:

The image has:

  • .border: Adds a default border.
  • .border-primary: Sets the border color to primary blue.
  • .border-3: Sets the border width to 3px.
  • .rounded-circle: Makes the image perfectly round.
  • .d-block .mx-auto: Centers the image horizontally.
