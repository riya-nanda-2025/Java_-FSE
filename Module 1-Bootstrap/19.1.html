Create a project directory: Bash mkdir my-bootstrap-sass-project cd my-bootstrap-sass-project Initialize npm: Bash npm init -y Install Bootstrap and Sass: Bash npm install bootstrap@5.3.0 sass This installs Bootstrap (which includes its Sass source files) and the Sass compiler. Create your Sass file: Create a file named scss/custom.scss (or similar) in your project: my-bootstrap-sass-project/ ├── node_modules/ ├── scss/ │ └── custom.scss └── index.html Add content to custom.scss: In scss/custom.scss, import Bootstrap's functions, variables, and then the rest of Bootstrap. Compile Sass: Open your terminal in the project root and run: Bash npx sass scss/custom.scss css/style.css This will compile custom.scss into css/style.css. For watching changes during development: Bash npx sass --watch scss/custom.scss css/style.css
